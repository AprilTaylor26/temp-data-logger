# Temperature Data Logger (Python)

## Overview
This project is a command-line application built in Python that collects, validates, transforms, and stores structured temperature data into a CSV file.

The application demonstrates foundational data engineering concepts including user input validation, data transformation, file-based persistence, and basic reporting.

---

## Features

- Interactive CLI menu system
- Input validation and exception handling
- Temperature conversion (Fahrenheit to Celsius)
- CSV file storage for structured data persistence
- Timestamp logging for data entry tracking
- File reading and display functionality

---

## Technologies Used

- Python 3
- Built-in libraries:
  - `datetime`
  - file I/O operations

---

## How It Works

1. User selects an option from the menu.
2. Temperature data and dates are entered.
3. Data is validated and converted.
4. Records are appended to a CSV file.
5. Users can view stored records.
